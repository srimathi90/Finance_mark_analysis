# Finance Market Analysis

## 📌 Project Description
This project analyzes financial market data using Python. It provides insights into trends, patterns, and predictions using data visualization and analysis techniques.

## 📂 Files Included
- streamlite_financial.py → Main application file
- final dashboard.ipynb → Jupyter notebook for analysis
- dataset (.csv file) → Financial data used in the project

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Streamlit
- Matplotlib

## ▶️ How to Run the Project
1. Install required libraries:
   pip install pandas numpy streamlit matplotlib

2. Run the Streamlit app:
   streamlit run streamlite_financial.py
